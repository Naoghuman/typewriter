PRo's Schreibmachine: 'Schnell schreiben'

Von 50an/min auf 300an/min in 1min 27.85sec. Können Sie mithalten?
Mit diesem Programm können Sie Ihre Schreibgeschwindigkeit trainieren, bzw. überprüfen.
Getestet werden die Buchstaben 'A'-'Z', 'Ö' sowie die Zeichen 'Komma' (','), 'Punkt' ('.') und das 'Leerzeichen' (' ').
Starten Sie das Programm mit einem Klick auf den Button 'Start', alternativ können Sie die Tasten 'Enter', 'F1' oder 'Strg' drücken.
Alle 2 Buchstaben erhöht sich die Geschwindigkeit von Anfangs 50an/min. auf maximal 300an/min..
Die Übung ist zu Ende, wenn die Buchstabenreihe die linke Seite erreicht oder nach einem Klick auf den Button 'Stop', alternativ können Sie die Tasten 'F2' oder 'Backspace' drücken um die Übung zu Beenden.
Allerdings wird eine vorzeitig beendete Übung nicht gewertet.
Nach der Eingabe Ihres Namens können Sie Ihre Leistung in einer Highscore-Tabelle begutachten.
Die Highscore-Tabelle können Sie ebenfalls durch Drücken von 'F5' aufrufen. Als kleiner Anreiz ist ein Highscore von mir zu sehen, der nicht gelöscht werden kann.

PS:
Geschrieben wird ohne die Benutzung der Shift-Taste, bzw. der Feststelltaste, da sonst das Zeichen 'ö' nicht erkannt wird.
